# templates/

本目录存放**可复制到其它项目的模板文件**。

## 当前模板

| 文件 | 用途 |
| --- | --- |
| `AGENTS.md` | 前后端通用的 Agent/开发者协作手册模板，覆盖技术栈、目录、规范、测试、Git、安全、Agent 硬约束等 |

## 使用方式

复制到目标仓库根目录后逐项替换 `<尖括号占位符>`：

```bash
cp templates/AGENTS.md <目标项目>/AGENTS.md
```

## 模板维护原则

1. 保持**可落地**：每节都要能直接写出命令或约束，避免空话
2. 保持**前后端对等**：两端约定同构（命名、测试、API 风格）
3. **硬约束单独成节**（见 AGENTS 模板第 11 节），便于 Agent 自动遵循
4. 模板不替代项目文档：项目特有细节写到 `docs/`，模板只给骨架
