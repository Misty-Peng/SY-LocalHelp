## 自动化注意事项
- 设计器 iframe 会在切换 Tab、模式、弹层后失效，需要重取 frame
- 某些选择器不是普通下拉，而是独立 iframe 弹窗
- `selectResourceData` 资源选择器使用 `el-table` + checkbox 语义，不能只点行文本
- 遇到遮罩层拦截时，页面内 DOM click 往往比常规 Playwright click 更稳定
- 对探索性操作，如不希望污染草稿，应在结束后刷新回滚
